Each question will have a hint provided to you if you are unsure where to start. Before you undertake the quiz, a few example questions have been provided. These will provide you with insight of how the quiz is structured and provide a basic example of how to search in Splunk. The example questions are as follows:

QUESTION 1:

How many simulation games were sold?


Step 1: To begin searching, first ensure that the filter at the top right has been set to “All time” and enter “*” in the search bar to display all results.



Step 2: By browsing the fields at the left-hand side, you can deduce what filter may be needed to narrow down the search.



Step 3: For this first question, the output can easily be found by selecting the “categoryID” filter and making “SIMULATION” its value. The Input should look this this:



Something



Answer: 273

QUESTION 2:

What is the first IP address that gave an incorrect password?


This question requires the use of the pipe symbol. Firstly, you must search for all entries pertaining to failed passwords, this can be carried out using “fail* password”. After carrying out this search, it can be deduced that the output is in descending order therefore to identify the first address more easily, a pipe can be used followed by “reverse” which will put the output into ascending order. Answer: 118.142.68.222

QUESTION 3:

What is the number of accesses for GET from any IP address which starts with 81?
